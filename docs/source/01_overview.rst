.. what the demo is, how to build, what it shows

Overview
========

This demo illustrates how IEC 61508 lifecycle artifacts
can be managed as structured objects using Sphinx-Needs.

Example:
A SIL2 Emergency Stop (E-Stop) safety function.

The demo demonstrates:

- Structured safety artifacts
- Lifecycle phase tagging
- Traceability
- Automated completeness checks
- Graph-based assurance visualization
